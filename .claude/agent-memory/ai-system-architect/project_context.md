This is the CodePath AI110 Final Project, due April 2026. The goal is to extend the existing PawPal+ Streamlit prototype into PawPal++: a production-quality AI-powered pet care app.

**Why:** Fulfills the AI110 final project rubric, which requires: a substantial AI feature (RAG, agentic workflow, fine-tuning, or reliability harness), a system architecture diagram, end-to-end functionality, reliability/guardrails, and a polished React+Vite+Tailwind+Shadcn UI.

**How to apply:** Every architectural and scope decision should be weighed against the rubric criteria. The project must be demonstrable end-to-end. The AI feature must be integrated, not a standalone demo. Python drives AI logic; React drives UI; a clear bridge connects them.

Key constraints from specs:
- Vite + React frontend, Shadcn + Tailwind for styling
- Python backend for AI logic (Google Gemini API)
- Light/dark/auto theme support, accessibility settings
- Footer: "PawPal++ - Where AI insights meet pet care", CodePath AI110 Final Project, Krish A. Patel + Claude Code, April 2026, Powered by Google Gemini
- Data persists in JSON (same as prototype)
- The existing pawpal_system.py logic is reusable as-is
